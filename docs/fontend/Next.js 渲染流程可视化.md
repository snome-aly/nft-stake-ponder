## 流程 A：首次访问（完整 SSR）

```
┌─────────────┐
│  1. 浏览器   │  发送请求：GET http://localhost:3000
└──────┬──────┘
       │
       ▼
┌─────────────────────────────────────────────────────────────┐
│  2. Next.js 服务器                                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 2.1 执行 Server Components                           │   │
│  │     ├─ async/await 数据获取                          │   │
│  │     └─ 生成初始 React 元素树                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 2.2 执行 Client Components（预渲染）                  │   │
│  │     ├─ 执行组件函数                                   │   │
│  │     ├─ 调用 useState（获取初始值）                    │   │
│  │     ├─ 注册 useEffect（不执行回调）                   │   │
│  │     └─ 生成 HTML 字符串                              │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 2.3 组装完整 HTML                                     │   │
│  │     ├─ 嵌入完整页面内容                               │   │
│  │     ├─ 添加 <script> 标签                            │   │
│  │     │   ├─ main-app.js (React runtime)              │   │
│  │     │   ├─ layout.js (Layout 组件)                   │   │
│  │     │   └─ page.js (页面组件)                        │   │
│  │     └─ 添加 next-themes 阻塞脚本                     │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │ 返回 HTML (200KB)
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  3. 浏览器接收 HTML                                          │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.1 解析 HTML                                        │   │
│  │     ├─ 构建 DOM 树                                   │   │
│  │     ├─ 构建 CSSOM 树                                 │   │
│  │     └─ 发现 <script> 标签                            │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.2 渲染页面                                          │   │
│  │     └─ 用户看到内容 ✨ (TTFB ~50ms)                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓ (并行)                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.3 下载 JavaScript                                   │   │
│  │     ├─ main-app.js (50KB)                            │   │
│  │     ├─ layout.js (30KB)                              │   │
│  │     └─ page.js (20KB)                                │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │ JS 下载完成
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  4. React Hydration                                          │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.1 执行所有组件（客户端首次渲染）                      │   │
│  │     ├─ 执行 Server Components                        │   │
│  │     ├─ 执行 Client Components                        │   │
│  │     │   ├─ useState(false) → mounted = false        │   │
│  │     │   ├─ useEffect → 注册到队列                    │   │
│  │     │   └─ if (!mounted) return null                │   │
│  │     └─ 生成虚拟 DOM                                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.2 对比虚拟 DOM 和真实 DOM                           │   │
│  │     虚拟: <div>Hello</div>                           │   │
│  │     真实: <div>Hello</div>                           │   │
│  │     结果: ✅ 匹配                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.3 添加事件监听器                                     │   │
│  │     ├─ onClick → 添加到按钮                          │   │
│  │     ├─ onChange → 添加到输入框                       │   │
│  │     └─ onSubmit → 添加到表单                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.4 Hydration 完成 ✨                                 │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  5. useEffect 执行                                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.1 执行 useEffect 队列中的回调                        │   │
│  │     useEffect(() => {                                │   │
│  │       setMounted(true)  ← 触发状态更新                │   │
│  │     }, [])                                           │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.2 触发重新渲染                                       │   │
│  │     ├─ 组件函数再次执行                               │   │
│  │     ├─ mounted = true                                │   │
│  │     ├─ 返回实际内容（不是 null）                      │   │
│  │     └─ React 更新 DOM                                │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.3 用户看到更新后的内容 ✨                            │   │
│  │     （如主题切换按钮出现）                             │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
              ╔═══════════════════╗
              ║  应用变成 SPA      ║
              ║  可以客户端导航     ║
              ╚═══════════════════╝
```

## 流程 B：客户端导航（Link 点击）

```
┌─────────────┐
│  用户操作    │  点击 <Link href="/debug">Debug</Link>
└──────┬──────┘
       │
       ▼
┌─────────────────────────────────────────────────────────────┐
│  1. Next.js Router (客户端)                                  │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 1.1 拦截点击事件                                       │   │
│  │     ├─ preventDefault()                              │   │
│  │     └─ 检查是否为外部链接                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 1.2 检查路由缓存                                       │   │
│  │     /debug 是否在缓存中？                             │   │
│  │     ├─ ✅ 在缓存且未过期 → 直接使用，跳到步骤 3       │   │
│  │     └─ ❌ 不在或过期 → 继续下一步                     │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  2. 请求服务器                                               │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 2.1 发送请求                                          │   │
│  │     GET /debug                                       │   │
│  │     Headers: {                                       │   │
│  │       'RSC': '1',  ← 关键！                          │   │
│  │       'Next-Router-State-Tree': '...'                │   │
│  │     }                                                │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  3. Next.js 服务器                                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.1 检测到 RSC 请求                                   │   │
│  │     └─ 不返回完整 HTML，返回 RSC Payload             │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.2 执行 app/debug/page.tsx                          │   │
│  │     ├─ Server Components                             │   │
│  │     └─ Client Components（预渲染）                    │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3.3 序列化成 RSC Payload                             │   │
│  │     {                                                │   │
│  │       "type": "div",                                 │   │
│  │       "props": {...},                                │   │
│  │       "chunks": ["debug-abc.js"],                    │   │
│  │       "data": {...}                                  │   │
│  │     }                                                │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │ 返回 Payload (5-10KB)
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  4. 客户端接收 Payload                                       │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.1 解码 RSC Payload                                 │   │
│  │     └─ 解析序列化的组件树                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.2 检查是否需要新的 JS                               │   │
│  │     chunks: ["debug-abc.js"]                         │   │
│  │     ├─ 已加载 → 跳过                                  │   │
│  │     └─ 未加载 → 异步加载                              │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓ (如果需要)                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4.3 加载 debug-abc.js                                │   │
│  │     GET /_next/static/chunks/debug-abc.js            │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────┐
│  5. React 渲染                                               │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.1 根据 Payload 生成新的虚拟 DOM                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.2 对比新旧虚拟 DOM                                  │   │
│  │     旧: <Home />                                     │   │

│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.3 计算最小 DOM 操作                                 │   │
│  │     ├─ <Header /> → 保持不变 ✅                       │   │
│  │     ├─ <main> → 替换内容 🔄                          │   │
│  │     └─ <Footer /> → 保持不变 ✅                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.4 更新 DOM                                         │   │
│  │     └─ 只更新 <main> 里的内容                        │   │
│  └─────────────────────────────────────────────────────┘   │
│                         ↓                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5.5 更新 URL                                         │   │
│  │     / → /debug                                       │   │
│  │     (使用 History API，不刷新页面)                    │   │
│  └─────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────┘
                         │
                         ▼
              ╔═══════════════════╗
              ║  导航完成 ✨       ║

              ╚═══════════════════╝
```

## SwitchTheme 组件的完整生命周期

```
┌──────────────────────────────────────────────────────────────┐
│                  SwitchTheme 组件生命周期                      │
└──────────────────────────────────────────────────────────────┘

阶段 1: 服务端渲染
─────────────────────────────────────────────────────────────
  function SwitchTheme() {
    const [mounted, setMounted] = useState(false)  // ✅ false
    const { resolvedTheme } = useTheme()           // ✅ undefined

    useEffect(() => {
      setMounted(true)  // ❌ 不执行
    }, [])

    if (!mounted) return null  // ✅ 返回 null
  }

  输出: null (空)
─────────────────────────────────────────────────────────────

  ↓ 服务端生成的 HTML

<nav>
  <button>Connect</button>
  <!-- SwitchTheme 位置：空 -->
</nav>

  ↓ 浏览器接收并渲染

阶段 2: 客户端 Hydration（第 1 次渲染）
─────────────────────────────────────────────────────────────
  function SwitchTheme() {
    const [mounted, setMounted] = useState(false)  // ✅ false
    const { resolvedTheme } = useTheme()           // ✅ "dark"

    useEffect(() => {
      setMounted(true)  // ⏳ 注册到队列
    }, [])

    if (!mounted) return null  // ✅ 返回 null
  }

  虚拟 DOM: null
  真实 DOM: 空
  ✅ 匹配！Hydration 成功
─────────────────────────────────────────────────────────────

  ↓ Hydration 完成

阶段 3: useEffect 执行
─────────────────────────────────────────────────────────────
  useEffect(() => {
    setMounted(true)  // ✅ 执行！
  }, [])

  触发状态更新 → React 调度重新渲染
─────────────────────────────────────────────────────────────

  ↓ 重新渲染

阶段 4: 客户端更新（第 2 次渲染）
─────────────────────────────────────────────────────────────
  function SwitchTheme() {
    const [mounted, setMounted] = useState(false)  // ✅ true
    const { resolvedTheme } = useTheme()           // ✅ "dark"
    const isDarkMode = resolvedTheme === "dark"    // ✅ true

    if (!mounted) return null  // ❌ 跳过

    return (  // ✅ 返回实际组件
      <div>
        <input checked={isDarkMode} />  // checked={true}
        <MoonIcon />  // 显示月亮图标
      </div>
    )
  }

  React 对比:
    旧虚拟 DOM: null
    新虚拟 DOM: <div><input .../><MoonIcon /></div>

  React 操作:
    在 <nav> 中插入新元素
─────────────────────────────────────────────────────────────

  ↓ DOM 更新完成

<nav>
  <button>Connect</button>
  <div>  ← 新插入的！
    <input type="checkbox" checked />
    <label><svg>...</svg></label>
  </div>
</nav>

  ↓

╔═══════════════════════════════════════╗
║  用户看到主题切换按钮出现 ✨          ║
╚═══════════════════════════════════════╝
```

## 时间轴对比

### 首次访问

```
0ms   ████████████████ 用户请求
      │
50ms  ████████████████ 服务器返回 HTML
      │                 └─ 用户看到内容 ✨ (TTFB)
      │
100ms ████████████████ JS 下载开始
      │
200ms ████████████████ JS 下载完成
      │
250ms ████████████████ Hydration 开始
      │
300ms ████████████████ Hydration 完成
      │                 └─ 添加事件监听器
      │
301ms ████████████████ useEffect 执行
      │
310ms ████████████████ DOM 更新完成
      │                 └─ 完全可交互 ✨
```

### 客户端导航

```
0ms   ████████████████ 用户点击 Link
      │
10ms  ████████████████ 请求 RSC Payload
      │
60ms  ████████████████ 接收 Payload
      │
80ms  ████████████████ React 渲染
      │
100ms ████████████████ DOM 更新完成
      │                 └─ 用户看到新页面 ✨
```

**对比：** 客户端导航比首次访问快 3 倍！

## 数据流动图

```
┌──────────────────────────────────────────────────────────────┐
│                       数据流动方向                             │
└──────────────────────────────────────────────────────────────┘

服务端
  │
  │  async function fetchData() {
  │    const data = await db.query(...)
  │    return data
  │  }
  │
  ▼
Server Component
  │
  │  export default async function Page() {
  │    const data = await fetchData()  ← 服务端数据
  │    return <ClientComponent data={data} />
  │  }
  │
  │ (通过 props 传递)
  │
  ▼
Client Component
  │
  │  "use client"
  │  export default function ClientComponent({ data }) {
  │    const [count, setCount] = useState(0)
  │    return (
  │      <div>
  │        {data.title} - {count}  ← 结合服务端和客户端数据
  │        <button onClick={() => setCount(count + 1)}>
  │      </div>
  │    )
  │  }
  │
  ▼
用户交互
```
